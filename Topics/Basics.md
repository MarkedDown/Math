
# Basics

*How to add **Latex** blocks.*

<br>

## Inline Block

*Useful for inline formulas as well as* <br>
*the only way to left align Latex blocks.*

<br>

```Markdown
$<Ｌａｔｅｘ　Ｃｏｎｔｅｎｔ>$
```

```Markdown
$x = 1 + a$
```

$x = 1 + a$

<br>
<br>

## $$ Block

*Similar to the previous block but multiline.*

```Markdown
$$<
Ｌａｔｅｘ　Ｃｏｎｔｅｎｔ
>$$
```

```Markdown
$$x = 1 + a$$
```

$$x = 1 + a$$
