## Syntax

<br>

### Basic

A red $\color{Red} word.$

```Markdown
A red $\color{Red} word.$
```

<br>

### Normalized

$\textrm{A red \color{Red} word}.$

```Markdown
$\textrm{A red \color{Red} word}.$
```
